Integrated Analysis of Hub Genes in TNBC

This project focuses on identifying key hub genes and analyzing intrinsically disordered regions in Triple-Negative Breast Cancer (TNBC) using bioinformatics approaches.

Objective
- Identify differentially expressed genes (DEGs)
- Perform pathway enrichment analysis
- Construct protein-protein interaction (PPI) networks
- Identify hub genes
- Visualize protein structures

Methods
- Datasets: GSE45827, GSE38959
- DEG analysis
- GO & KEGG enrichment (ShinyGO)
- PPI network (STRING, Cytoscape)
- Hub gene identification (cytoHubba)
- 3D visualization (PDB, PyMOL)

Key Results
- Hub genes: TTK, TOP2A, CCNA2
- Enriched in cell cycle pathways
